A simple mod, like ever second run `heal` command.

in `com.blackmoss.undeadexperiment.cfg` change the `HealCountdown` can change heal countdown, default is 1f.(1 second)

# 不死Exp
一个简单的模组，功能类似每秒运行一次`heal`命令。

在 `BepInEx/config/com.blackmoss.undeadexperiment.cfg` 里更改 `HealCountdown` 的值可以更改治疗的冷却，默认是1f（1秒）。
